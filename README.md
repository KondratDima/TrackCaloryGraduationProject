# TrackCalory - AI-асистент для підрахунку калорій

## Інформація про проект

**Тема:** Розробка кросплатформного мобільного додатку для автоматизованого підрахунку калорій із використанням технологій штучного інтелекту та комп'ютерного зору

**Автор:** Кондрат Дмитро Олексійович

**Керівники:**
Керівник з інформатики: Самарцов Миколайович

**Навчальний заклад:** Дніпровський науковий ліцей інформаційних технологій (ЛІТ)

**Рік:** 2025-2026

---
TrackCalory - це мобільний додаток для контролю харчування, який використовує Google Gemini Vision AI для автоматичного розпізнавання страв на фотографіях та визначення їх калорійності. Додаток скорочує час на підрахунок калорій з 10 хвилин до 10 секунд на один прийом їжі.

**Основні можливості:**
- AI-розпізнавання страв з фотографій
- Автоматичний розрахунок денної норми калорій
- Кросплатформність (Android, iOS, Windows)
- Локальне зберігання даних (повна конфіденційність)
- Статистика та історія харчування
- Підтримка українських страв

Мова програмування: C# 11
Версія фреймворку: .NET 7.0 / .NET MAUI
Платформи: Android, iOS, Windows

## ВИМОГИ ДО АПАРАТНИХ ПРИСТРОЇВ

### Мінімальні системні вимоги:
**Для Android:**
- Процесор: ARM або x86 архітектури, 1 ГГц або швидший
- Оперативна пам'ять: мінімум 2 ГБ (рекомендовано 4 ГБ)
- Вільний дисковий простір: 150 МБ для додатку + 50 МБ для бази даних
- Камера: будь-яка (для фотографування страв)
- Дисплей: роздільна здатність мінімум 720x1280 пікселів

**Для iOS:**
- Процесор: Apple A7 або новіший (64-bit)
- Оперативна пам'ять: мінімум 2 ГБ
- Вільний дисковий простір: 150 МБ для додатку + 50 МБ для бази даних
- Камера: будь-яка
- Дисплей: роздільна здатність мінімум 750x1334 пікселів

**Для Windows:**
- Процесор: Intel Core i3 або еквівалент, 1.5 ГГц або швидший
- Оперативна пам'ять: мінімум 4 ГБ (рекомендовано 8 ГБ)
- Вільний дисковий простір: 200 МБ для додатку + 100 МБ для .NET Runtime + 50 МБ для бази даних
- Дисплей: роздільна здатність мінімум 1024x768 пікселів

## ВІДОМОСТІ ПРО ОСОБЛИВОСТІ УСТАНОВКИ ТА РОБОТИ

### Установка:

**Android:**

1. Завантажити файл `TrackCalory.apk`
2. Увімкнути "Невідомі джерела" в налаштуваннях безпеки
3. Відкрити APK файл та натиснути "Встановити"
4. Надати дозволи на доступ до камери та файлової системи

**iOS:**

1. Встановити через TestFlight
АБО
2. Встановити через Xcode на підключений пристрій

**Windows:**

1. Розпакувати архів з додатком
2. Натиснути на сертифікат TrackCalory_0.0.2.0_x64 та обрати встановити сертифікат
3. Обрати "локальний комп'ютер" у меню імпорта сертифіката 
4. Критичний крок: Виберіть «Помістити всі сертифікати в наступне сховище» -> Огляд.
5. Виберіть папку Trusted People (Довірені особи).
6. Натисніть ОК, Далі та Готово.
7. Після цього запустить файл TrackCalory_0.0.2.0_x64.msix для встановлення додатку
